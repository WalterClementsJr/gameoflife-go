Conway Game of life in Go & Raylib.

# Development

```sh
# download dependencies
dnf install mesa-libGL-devel libXi-devel libXcursor-devel libXrandr-devel libXinerama-devel wayland-devel libxkbcommon-devel

go install

make run
```
